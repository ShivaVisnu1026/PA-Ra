# IO Pattern Scripts - 使用說明

## 📋 概述

這組腳本從 **Price Action All In One** 指標中提取了 IO 型態偵測功能，創建為三個獨立的腳本：

1. **IOPattern_Indicator.xs** - 指標版：在圖表上顯示 IO 型態
2. **IOPattern_Alert.xs** - 警示版：偵測到型態時發出警示
3. **IOPattern_Screener.xs** - 選股版：篩選符合 IO 型態的股票

這三個腳本可在**所有時間週期**使用（分鐘線、日線、週線、月線等）。

---

## 🎯 什麼是 IO 型態？

IO 型態是 Price Action 交易中重要的K棒組合型態，用於判斷市場狀態和潛在交易機會。

### 型態類型

#### 1️⃣ OO (Outside-Outside) - 連續外包
**特徵**：
- 第2根K棒完全包住第1根K棒
- 第3根K棒完全包住第2根K棒
- 連續兩次外包關係

**市場意義**：
- ✅ 波動擴大
- ✅ 可能是趨勢加速
- ✅ 或即將反轉的前兆

**交易策略**：
- 觀察突破方向
- 順勢交易
- 設定合理的止損位

```
  ┌─────┐
  │  3  │  ← 第3根包住第2根
  │ ┌─┐ │
  │ │2│ │  ← 第2根包住第1根
  │ ││││ │
  │ │─│ │  ← 第1根
  └─────┘
```

#### 2️⃣ II (Inside-Inside) - 連續內包
**特徵**：
- 第2根K棒完全被第1根K棒包住
- 第3根K棒完全被第2根K棒包住
- 連續兩次內包關係

**市場意義**：
- ✅ 盤整收斂
- ✅ 波動率降低
- ✅ 通常預示即將突破

**交易策略**：
- 等待突破確認
- 突破方向為交易方向
- 可設定較窄的止損

```
  ┌─────┐
  │ ┌─┐ │  ← 第1根
  │ │ │ │
  │ │┌┐│ │  ← 第2根被第1根包住
  │ │││││ │
  │ │└┘│ │  ← 第3根被第2根包住
  └─────┘
```

#### 3️⃣ IOI (Inside-Outside-Inside) - 內外內
**特徵**：
- 第2根K棒包住第1根（Outside）
- 第3根K棒被第2根包住（Inside）
- 先突破後回縮的組合

**市場意義**：
- ⚠️ 可能是假突破
- ⚠️ 或洗盤動作
- ⚠️ 市場猶豫不決

**交易策略**：
- 謹慎對待
- 等待更多確認
- 避免貿然進場

```
  ┌─────┐
  │  2  │  ← 第2根包住第1根
  │ ┌┐  │
  │ ││  │  ← 第1根
  │ │┌┐ │
  │ │└┘ │  ← 第3根被包住（回縮）
  └─────┘
```

## 📊 IOPattern_Indicator.xs - 指標使用說明

### 功能

1. **視覺化顯示** IO 型態
2. **計算型態強度**（弱/中/強）
3. **繪製圖形**供技術分析

### 參數設定

| 參數 | 預設值 | 說明 |
|------|--------|------|
| DetectOO | true | 偵測 OO 型態 |
| DetectII | true | 偵測 II 型態 |
| DetectIOI | true | 偵測 IOI 型態 |
| ShowLabels | true | 顯示型態標籤 |
| ShowArrows | true | 顯示箭頭標記 |

### 繪圖輸出

| Plot | 內容 | 數值說明 |
|------|------|----------|
| Plot1 | IO型態類型 | 0=無, 1=II, 2=OO, 3=IOI |
| Plot2 | 型態強度 | 0=無, 1=弱, 2=中, 3=強 |
| Plot3 | II型態標記 | 出現時在K棒上方顯示 |
| Plot4 | OO型態標記 | 出現時在K棒上方顯示 |
| Plot5 | IOI型態標記 | 出現時在K棒上方顯示 |

### 強度判斷標準

#### OO 型態強度
- **強(3)**: 當前K棒範圍 ≥ 前一根的 1.5 倍
- **中(2)**: 當前K棒範圍 ≥ 前一根的 1.2 倍
- **弱(1)**: 其他情況

#### II 型態強度
- **強(3)**: 當前K棒範圍 ≤ 前一根的 0.5 倍（收斂>50%）
- **中(2)**: 當前K棒範圍 ≤ 前一根的 0.7 倍（收斂30-50%）
- **弱(1)**: 其他情況

#### IOI 型態強度
- **中(2)**: IOI 一律視為中等強度

### 使用方式

1. **載入指標**
   ```
   在XQ全球贏家中：
   技術分析 → 新增指標 → 載入 IOPattern_Indicator.xs
   ```

2. **調整參數**
   - 根據交易風格選擇要偵測的型態
   - 日內交易建議關注 II 和 OO
   - 波段交易關注所有型態

3. **觀察圖表**
   - Plot1: 顯示型態類型數值
   - Plot2: 顯示型態強度數值
   - Plot3-5: 在K棒上方顯示型態標記

4. **配合選股**
   - 使用 IOPattern_Screener.xs 進行型態選股
   - 篩選符合條件的股票清單

## 🔔 IOPattern_Alert.xs - 警示使用說明

### 功能

1. **即時監控** IO 型態出現
2. **發送警示**通知交易者
3. **過濾條件**減少雜訊

### 參數設定

| 參數 | 預設值 | 說明 |
|------|--------|------|
| DetectOO | true | 警示 OO 型態 |
| DetectII | true | 警示 II 型態 |
| DetectIOI | true | 警示 IOI 型態 |
| MinStrength | 1 | 最小型態強度（1=弱/2=中/3=強） |
| AlertOncePerBar | true | 每根K棒只警示一次 |
| UseVolumeFilter | false | 使用成交量過濾 |
| MinVolumeRatio | 1.0 | 最小量比（相對20日均量） |

### 警示訊息格式

```
【IO型態警示】股票代碼 偵測到 型態名稱 (強度:X) 量比:X.XX 時間:HHMM 價:XX.XX
```

**範例**：
```
【IO型態警示】2330 偵測到 II(內包-內包) (強度:強) 量比:1.45 時間:1330 價:585.00
【IO型態警示】0050 偵測到 OO(外包-外包) (強度:中) 量比:1.12 時間:1015 價:138.50
【IO型態警示】2454 偵測到 IOI(內-外-內) (強度:中) 時間:1425 價:425.50
```

### 使用方式

1. **設定警示**
   ```
   在XQ全球贏家中：
   警示系統 → 新增警示 → 載入 IOPattern_Alert.xs
   ```

2. **選擇監控標的**
   - 可監控單一股票
   - 或設定為全市場掃描

3. **調整參數**
   - **日內交易者**：MinStrength=2，減少雜訊
   - **波段交易者**：MinStrength=1，捕捉所有訊號
   - **高勝率策略**：UseVolumeFilter=true, MinVolumeRatio=1.2

4. **接收通知**
   - 彈窗通知
   - 聲音提醒
   - 推播至手機（如有設定）

## 📊 IOPattern_Screener.xs - 選股使用說明

### 功能

1. **批次掃描**全市場股票
2. **篩選符合** IO 型態的股票
3. **輸出欄位**供排序和進階篩選

### 參數設定

| 參數 | 預設值 | 說明 |
|------|--------|------|
| DetectOO | true | 選出 OO 型態 |
| DetectII | true | 選出 II 型態 |
| DetectIOI | true | 選出 IOI 型態 |
| MinStrength | 1 | 最小型態強度（1=弱/2=中/3=強） |
| UseVolumeFilter | false | 使用成交量過濾 |
| MinVolumeRatio | 1.0 | 最小量比（相對20日均量） |
| SortByStrength | true | 依型態強度排序 |

### 輸出欄位

| 欄位 | 內容 | 數值說明 |
|------|------|----------|
| OutputField1 | IO型態 | 0=無, 1=II, 2=OO, 3=IOI |
| OutputField2 | 型態強度 | 0=無, 1=弱, 2=中, 3=強 |
| OutputField3 | OO型態 | 0=否, 1=是 |
| OutputField4 | II型態 | 0=否, 1=是 |
| OutputField5 | IOI型態 | 0=否, 1=是 |
| OutputField6 | 量比 | 當日量/20日均量 |
| OutputField7 | 收盤價 | 最新收盤價 |

### 使用方式

1. **執行選股**
   ```
   在XQ全球贏家中：
   選股中心 → 新增選股 → 載入 IOPattern_Screener.xs
   ```

2. **調整篩選條件**
   - **尋找突破機會（II型態）**：
     - DetectII=true, 其他=false
     - MinStrength=2
     - UseVolumeFilter=true
   
   - **尋找趨勢加速（OO型態）**：
     - DetectOO=true, 其他=false
     - MinStrength=2
     - MinVolumeRatio=1.2
   
   - **尋找所有型態**：
     - 全部=true
     - MinStrength=2（減少雜訊）

3. **查看結果**
   - 依型態強度排序（強→弱）
   - 或依型態類型排序（OO→IOI→II）
   - 可進一步用量比篩選

4. **後續動作**
   - 點選個股查看圖表（使用 IOPattern_Indicator.xs）
   - 設定警示監控（使用 IOPattern_Alert.xs）
   - 評估進場時機

## 💡 實戰應用策略

### 策略 1：II 型態突破交易（短線）

**適用時間週期**：5分鐘、15分鐘

**交易邏輯**：
1. 等待 II 型態出現（連續內包）
2. 觀察第4根K棒突破方向
3. 突破向上→做多，突破向下→做空
4. 止損設在型態的另一端

**參數設定**：
```
IOPattern_Alert.xs:
- DetectII = true
- DetectOO = false
- DetectIOI = false
- MinStrength = 2 (中等強度以上)
- UseVolumeFilter = true
- MinVolumeRatio = 1.2
```

**優勢**：
- ✅ 明確的進場訊號
- ✅ 窄止損範圍
- ✅ 突破後通常有較大波動

### 策略 2：OO 型態趨勢加速（波段）

**適用時間週期**：日線、60分鐘

**交易邏輯**：
1. 在明確趨勢中等待 OO 型態
2. OO 型態確認趨勢加速
3. 順勢進場
4. 使用移動止損

**參數設定**：
```
IOPattern_Alert.xs:
- DetectOO = true
- DetectII = false
- DetectIOI = false
- MinStrength = 2
- UseVolumeFilter = true
- MinVolumeRatio = 1.5
```

**優勢**：
- ✅ 趨勢加速點進場
- ✅ 較大獲利空間
- ✅ 配合趨勢勝率高

### 策略 3：IOI 型態反轉交易（進階）

**適用時間週期**：15分鐘、60分鐘、日線

**交易邏輯**：
1. 在關鍵支撐/壓力位出現 IOI
2. 判斷為假突破
3. 反向進場
4. 嚴格止損

**參數設定**：
```
IOPattern_Alert.xs:
- DetectIOI = true
- DetectOO = false
- DetectII = false
- MinStrength = 2
- UseVolumeFilter = false (IOI本身已篩選)
```

**注意**：
- ⚠️ 需配合支撐壓力判斷
- ⚠️ 風險較高，建議進階交易者使用
- ⚠️ 需要嚴格止損

## 📈 不同時間週期的應用

### 1分鐘 / 5分鐘線
**特性**：訊號頻繁，雜訊多  
**建議**：
- MinStrength = 2 或 3
- 使用成交量過濾
- 適合當沖交易者

### 15分鐘 / 30分鐘線
**特性**：訊號與雜訊比適中  
**建議**：
- MinStrength = 2
- 適合日內波段
- 最推薦的時間週期

### 60分鐘線
**特性**：訊號較少但可靠  
**建議**：
- MinStrength = 1
- 適合日內到隔日波段
- 配合日線趨勢判斷

### 日線
**特性**：訊號稀少但重要  
**建議**：
- MinStrength = 1
- 捕捉所有型態
- 適合波段交易
- 每個訊號都值得重視

### 週線 / 月線
**特性**：訊號極少但極為重要  
**建議**：
- MinStrength = 1
- 不使用成交量過濾
- 適合長線投資
- 出現訊號時仔細分析

## 🔄 與其他指標搭配

### 搭配趨勢指標

#### 配合 EMA
```
趨勢確認：
- 價格在 EMA20 之上 + II型態 = 多頭突破買進
- 價格在 EMA20 之下 + II型態 = 空頭突破放空
```

#### 配合 MACD
```
動能確認：
- MACD 金叉 + OO型態 = 趨勢加速做多
- MACD 死叉 + OO型態 = 趨勢加速做空
```

### 搭配支撐壓力

```
關鍵位置的 IO 型態更重要：
- 前高/前低出現 II = 突破機會大
- 重要支撐/壓力出現 IOI = 假突破可能性高
```

### 搭配成交量

```
量價配合：
- II型態突破 + 爆量 = 有效突破
- OO型態 + 量能遞增 = 趨勢持續
```

## ⚠️ 注意事項

### 1. 型態的有效性
- ❌ 不是每個 IO 型態都會導致明確走勢
- ✅ 需要配合其他技術分析工具
- ✅ 在趨勢中的型態較可靠

### 2. 時間週期的選擇
- 🎯 選擇符合你交易風格的週期
- 🎯 較長週期的訊號較可靠
- 🎯 不建議同時監控太多週期

### 3. 風險控制
- 🛡️ 每個交易都要設定止損
- 🛡️ II 型態突破失敗要快速止損
- 🛡️ OO 型態可能是反轉前兆，注意防守

### 4. 市場環境
- 📊 趨勢市場：OO 和 II 較有效
- 📊 盤整市場：IOI 較常出現
- 📊 避免在重大新聞前後交易型態

   - IOPattern_Screener.xs（選股）


   - 指標：在圖表上新增指標，應該能看到 IO 型態標記
   - 警示：設定警示，等待型態出現時應收到通知
   - 選股：執行選股，應該能列出符合型態的股票

### 建議設定組合

#### 日內交易者設定
```
時間週期：5分鐘 + 15分鐘
指標：兩個週期都掛 IOPattern_Indicator.xs
警示：15分鐘掛 IOPattern_Alert.xs (MinStrength=2)
選股：不使用（標的已確定）

```

#### 波段交易者設定

```
時間週期：日線 + 60分鐘

警示：日線和60分鐘都掛 IOPattern_Alert.xs (MinStrength=1)
選股：每日收盤後執行 IOPattern_Screener.xs (MinStrength=2)
目標：捕捉中期波段機會
```

#### 長線投資者設定
```
時間週期：週線 + 月線
指標：週線掛 IOPattern_Indicator.xs
警示：週線掛 IOPattern_Alert.xs (MinStrength=1)
```

## 🤝 技術支援

### 常見問題

**Q: 為什麼有時候看到型態但沒有警示？**  
A: 檢查以下設定：
- MinStrength 設定是否過高
- UseVolumeFilter 是否啟用且量能不足
- AlertOncePerBar 是否已觸發過

**Q: 不同時間週期的型態哪個更可靠？**  
A: 一般來說，較長時間週期（日線、週線）的型態更可靠，但機會也較少。建議根據你的交易週期選擇。

**Q: 可以只偵測強度為「強」的型態嗎？**  
A: 可以，設定 `MinStrength = 3` 即可，但訊號會變得非常稀少。

**Q: IOI 型態的成功率如何？**  
A: IOI 是較複雜的型態，需要配合支撐壓力判斷。單獨使用勝率較低，建議進階交易者在特定情境使用。
