# XScript 範例程式碼

> 精選的學習範例，從基礎到進階

---

## 📚 範例分類

### [01_basics/](01_basics/) - 基礎範例
適合初學者，包含 XScript 的基本語法和概念。

**內容**：
- 變數宣告和使用
- 基本運算和函數
- 條件判斷
- 簡單的指標計算

### [02_indicators/](02_indicators/) - 指標範例
技術指標的開發範例。

**內容**：
- 移動平均線（MA、EMA）
- 動量指標（RSI、MACD、KD）
- 波動率指標（布林通道、ATR）
- 自訂指標

### [03_alerts/](03_alerts/) - 警示範例
警示腳本的開發範例。

**內容**：
- 價格突破警示
- 指標超買超賣警示
- 型態識別警示（Pin Bar、吞噬等）
- 成交量異常警示
- 多重條件警示

### [04_screeners/](04_screeners/) - 選股範例
選股腳本的開發範例。

**內容**：
- 技術面選股
- 基本面選股
- 籌碼面選股
- 產業選股
- 綜合選股

### [05_trading/](05_trading/) - 交易範例
自動交易策略範例。

**內容**：
- 均線交叉策略
- 突破策略
- 反轉策略
- 風險管理範例
- 完整交易系統

## 🎯 如何使用範例

### 學習流程
1. **從基礎開始**：先看 `01_basics/` 了解語法
2. **選擇方向**：根據需求選擇對應分類
3. **研究範例**：閱讀程式碼和註解
4. **動手實作**：修改參數，觀察結果
5. **自己開發**：基於範例開發自己的腳本

### 閱讀建議
- 先看檔案開頭的說明註解
- 注意各區段的註解說明
- 理解為什麼這樣寫
- 嘗試修改參數和邏輯
- 參考 [docs/guides/](../docs/guides/) 的詳細說明

## 📝 範例規範

### 所有範例都包含
- ✅ 完整的開頭註解（名稱、類型、功能、作者等）
- ✅ 清晰的區段分隔
- ✅ 詳細的註解說明
- ✅ 有意義的變數命名
- ✅ 遵循 [程式碼風格指南](../docs/guides/STYLE_GUIDE.md)

### 範例特點
- **可直接執行**：複製貼上即可使用
- **註解豐富**：解釋每個步驟的用途
- **參數可調**：使用 input 定義可調整參數
- **結構清晰**：按照標準結構組織
