# 日內突破策略 - 15分鐘K棒（增強版）

## 📋 策略概述

這是原版日內突破策略的增強版本，在原有基礎上加入了EMA趨勢確認條件，提升策略的穩定性和勝率。

## 🎯 策略邏輯

### 信號棒識別條件（第一根K棒）

#### 基本條件
1. **收紅**：Close > Open
2. **漲幅確認**：相對前一根收盤價漲幅 ≥ 3%
3. **上影線控制**：上影線百分比 ≤ 2%
4. **實體確認**：實體漲幅百分比 ≥ 3%
5. **突破確認**：突破過去20根K棒的最高點
6. **流動性過濾**：日成交量 ≥ 1000張

#### 增強條件（新增）
7. **EMA趨勢確認**：日收盤價 > EMA8
8. **EMA排列確認**：EMA8 > EMA20
9. **EMA20遞增**：EMA20 > EMA20[1]

### 進場條件
- **時機**：K棒2-16之間
- **觸發**：收盤價由下往上突破信號棒高點+1tick
- **方式**：市價進場

### 出場條件
1. **時間平倉**：K棒16後強制平倉
2. **停損**：跌破信號棒低點
3. **停利**：達到風報比1:1目標
4. **隔日保險**：防止意外隔日持倉

## ⚙️ 參數設定

| 參數名稱 | 預設值 | 說明 |
|---------|--------|------|
| min_gain_percent | 3.0 | 最小漲幅百分比 |
| max_upper_shadow | 2.0 | 最大上影線百分比 |
| max_body_percent | 3.0 | 最小實體漲幅百分比 |
| lookback_bars | 20 | 回顧期數（檢查是否創新高） |
| risk_reward_ratio | 1.0 | 風報比 |
| min_daily_volume | 1000 | 最小日成交量（張） |
| ema8_period | 8 | EMA8期數 |
| ema20_period | 20 | EMA20期數 |

## 🔧 技術特點

### EMA趨勢確認機制
```xs
// 計算日頻率EMA（避免使用保留字首 daily）
value5 = GetField("Close", "D");
value6 = XAverage(value5, ema8_period);
value7 = XAverage(value5, ema20_period);

// EMA趨勢條件
condition7 = value5 > value6;                   // 日收盤價 > EMA8
condition8 = value6 > value7;                   // EMA8 > EMA20
condition9 = value7 > value7[1];                // EMA20遞增
```

### 跨頻率資料應用
- **日頻率成交量**：`GetField("Volume", "D")`
- **日頻率收盤價**：`GetField("Close", "D")`
- **EMA計算**：基於日頻率資料計算趨勢指標

## 📊 策略優勢

### 1. 趨勢確認
- **多層次確認**：結合K棒形態、突破、EMA趨勢
- **減少假信號**：EMA條件過濾掉逆勢突破
- **提升勝率**：只在明確上升趨勢中進場

### 2. 風險控制
- **時間控制**：當日平倉，避免隔夜風險
- **停損機制**：跌破信號棒低點立即停損
- **風報比控制**：固定1:1風報比

### 3. 流動性保障
- **成交量過濾**：確保標的有足夠流動性
- **市價進場**：確保突破時能快速成交

## 🎯 適用場景

### 適合的市場環境
- **上升趨勢市場**：EMA20遞增確認趨勢向上
- **突破行情**：適合捕捉關鍵阻力位突破
- **流動性充足**：日成交量1000張以上的標的

### 不適合的市場環境
- **盤整市場**：EMA20不遞增，缺乏趨勢確認
- **下降趨勢**：EMA排列不正確
- **低流動性標的**：成交量不足，容易滑價

## 📈 預期效果

### 相比原版的改進
1. **勝率提升**：EMA趨勢確認減少假突破
2. **穩定性增強**：多層次條件過濾
3. **風險降低**：只在明確趨勢中進場

### 可能的限制
1. **信號減少**：更嚴格的條件可能減少交易機會
2. **滯後性**：EMA確認可能錯過快速突破
3. **參數敏感**：需要根據市場調整EMA參數

## 🔍 使用建議

### 參數調整
- **EMA期數**：可根據標的特性調整（如5,13,21）
- **成交量門檻**：根據標的流動性調整
- **風報比**：可調整為1.5:1或2:1

### 監控重點
- **EMA排列**：確保EMA8 > EMA20且EMA20遞增
- **成交量**：確保日成交量符合門檻
- **突破確認**：確認真正的突破而非假突破
