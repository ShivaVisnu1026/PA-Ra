# 日內突破策略 - 15分鐘K棒

## 📚 策略概述

這是一個基於15分鐘K棒的日內突破交易策略，專門設計用於捕捉開盤後的強勢突破行情。

## 🎯 策略邏輯

### 信號棒識別
每天開盤後第一根15分鐘K棒需要符合以下條件：

1. **收紅**：收盤價高於開盤價
2. **強勢上漲**：相對前一根收盤價上漲超過3%
3. **上影線控制**：上影線不超過收盤價的2%
4. **實體夠大**：實體漲幅至少超過3%
5. **整理突破**：過去20根K棒內都沒有創新高
6. **成交量確認**：成交量放大至少20%（進階版）

### 進場條件
- 即時價格從下往上碰到信號棒高點+1tick
- 使用 `AddSpread` 函數精確計算跳動點
- 每天只允許做一次交易

### 出場條件
1. **停損**：價格跌破信號棒的低點
2. **停利**：達到風報比1:1的目標價位
3. **時間停損**：下午1:20後強制平倉（台灣股市收盤前10分鐘）
4. **緊急停損**：回撤超過5%時強制平倉（進階版）

## 💻 腳本版本

### 標準版 (`日內突破策略_15分鐘.xs`)
- 基本的突破策略邏輯
- 使用 `AddSpread` 函數處理 tick 計算
- 即時價格從下往上碰到信號棒高點+1tick 才觸發進場
- 固定停損停利設定
- 收盤前自動平倉（13:20）
- 修正了編譯錯誤

## ⚙️ 參數說明

### 基本參數
- **min_gain_percent**：最小漲幅百分比（預設3.0%）
- **max_upper_shadow**：最大上影線百分比（預設2.0%）
- **min_body_percent**：最小實體漲幅百分比（預設3.0%）
- **lookback_bars**：回顧期數（預設20根）
- **risk_reward_ratio**：風報比（預設1.0）
### 進階參數
- **close_time**：強制平倉時間（預設1320，台灣股市收盤前10分鐘）

## 📊 監控數據

### 監控數據
- 信號棒高點和低點
- 進場價格
- 停損停利價格
- 交易狀態

## 🎯 策略特點

### 優勢
1. **明確的進場條件**：基於技術分析的明確信號
2. **風險控制**：固定停損和風報比控制
3. **時間管理**：收盤前自動平倉避免隔夜風險
4. **適合日內交易**：專門設計用於15分鐘週期

### 適用市場
- 流動性良好的股票
- 波動性適中的市場
- 有明確趨勢的交易日

### 注意事項
1. **僅適用15分鐘週期**：策略設計專門針對15分鐘K棒
2. **需要流動性**：確保有足夠的成交量支持突破
3. **市場環境**：在趨勢明確的市場中效果較佳
4. **風險管理**：嚴格執行停損停利規則

## 🔧 使用建議

### 參數調整
- 根據不同股票的波動特性調整漲幅參數
- 根據市場環境調整風報比
- 根據交易時間調整強制平倉時間

### 風險控制
- 設定合理的資金管理規則
- 避免在重大消息發布前交易
- 注意市場流動性變化

### 監控重點
- 關注信號棒的品質
- 監控進場後的價格行為
- 及時調整停損停利設定

## 📈 實戰應用

### 適用場景
1. **開盤強勢**：開盤後出現明顯的強勢上漲
2. **整理突破**：經過一段時間整理後的突破
3. **成交量配合**：突破時有成交量放大

### 避免場景
1. **震盪市場**：在橫盤震盪的市場中效果較差
2. **低流動性**：成交量不足的股票
3. **重大消息**：有重大消息發布的交易日

## 🚀 優化方向

### 策略優化
1. **動態參數**：根據市場波動性動態調整參數
2. **多時間框架**：結合其他時間框架的確認
3. **市場環境識別**：根據市場環境調整策略參數

### 風險管理優化
1. **資金管理**：根據帳戶資金動態調整部位大小
2. **相關性控制**：避免同時持有多個相關性高的部位
3. **回撤控制**：設定整體回撤限制

這個策略提供了一個完整的日內突破交易框架，可以根據實際需求進行調整和優化。
