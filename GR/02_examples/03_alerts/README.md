# XScript 警示腳本範例

本資料夾包含各種 XScript 警示腳本的範例。

## RSI 超過 70 警示腳本

### 1. 簡化版 (`RSI_超過70警示_簡化版.xs`)
- **用途**：最簡單的 RSI 超買警示
- **功能**：當 RSI(14) 超過 70 時發出警示
- **適用對象**：初學者
- **程式碼行數**：6 行

### 2. 標準版 (`RSI_超過70警示.xs`)
- **用途**：標準的 RSI 超買警示
- **功能**：
  - 可調整 RSI 計算期數（預設 14）
  - 可調整警示閾值（預設 70）
  - 可選擇計算價格（預設收盤價）
- **適用對象**：一般使用者
- **程式碼行數**：25 行

### 3. 進階版 (`RSI_超過70警示_進階版.xs`)
- **用途**：進階的 RSI 超買警示
- **功能**：
  - 避免重複警示（RSI 持續在高檔時不會一直觸發）
  - 可設定 RSI 回落至一定數值後重新啟用警示
  - 顯示當前 RSI 數值
  - 顯示警示狀態
- **適用對象**：進階使用者
- **程式碼行數**：45 行

## 使用說明

### 如何選擇版本
1. **初學者**：使用簡化版，程式碼簡單易懂
2. **一般使用者**：使用標準版，功能完整且可調整參數
3. **進階使用者**：使用進階版，功能最完整，避免重複警示

### 參數說明
- **RSI_Length**：RSI 計算期數，通常使用 14
- **RSI_Threshold**：RSI 警示閾值，通常使用 70（超買）或 30（超賣）
- **RSI_Reset**：RSI 重新啟用警示的數值，建議設為 65
- **RSI_Price**：RSI 計算價格，通常使用 Close（收盤價）

### 技術分析說明
- **RSI > 70**：表示股價可能進入超買區，有回檔風險
- **RSI < 30**：表示股價可能進入超賣區，有反彈機會
- **RSI = 50**：表示股價處於中性區域
