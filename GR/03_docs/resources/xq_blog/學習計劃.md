### **Group 語法範例**
```xs
Group: _group();//宣告群組
var: _i(0), _size(0);
var: _Health(0);
_group = GetSymbolList("細產業名稱");
_size = GetSymbolListSize(_group);

for _i = 0 to _size - 1 do
begin
    // 處理每個成份股
    _Health += GetField("營業利益", "D");
end;
```
