# XQ官方部落格學習資源

## 📚 資源概述

## 🔧 技術特色

### **Group 語法應用**
官方部落格大量使用 Group 語法來處理細產業分析：

```xs
Group: _group();//宣告群組
var: _i(0), _size(0);
_group = GetSymbolList("細產業名稱");
_size = GetSymbolListSize(_group);

for _i = 0 to _size - 1 do
begin
    // 處理每個成份股
end;
```
