    選股欄位 - 價格

    欄位名稱 欄位說明 支援商品
    Jensen Jensen指標用於衡量實際的投資回報超過基於其風險水平所預期的市場回報。 台股 美(ETF)
    SHARPE Sharpe指標衡量投資相對於其風險所帶來的超額回報率，即投資回報超過無風險利率部分的表現。 台股 美(ETF)
    Treynor Treynor指標衡量投資相對於系統性風險的回報率。 台股 美(ETF)
    下游股價指標 利用客戶的股價波動計算股價指數。 台股
    上游股價指標 利用供應商的股價波動計算股價指數。 台股
    月平均收益率 最近21日的平均賺賠比例。 台股
    本益比 股票的本益比。 台股 美(股票)
    同業股價指標 利用同業競爭公司的股價波動計算股價指數。 台股
    收盤價 K棒資料的收盤價(最後一個成交價位)。 台股 香港股票 大陸股票 美(股票)
    均價 每日的平均成交價。 台股 大盤 類股指數 香港股票 大陸股票
    投資建議目標價 市場機構對個股的股價預估平均值。
    美股股票才適用本欄位。    		 美(股票)
    貝他值 個股的股價變動與市場大盤指數變動的相關性。
    也就是當大盤平均上漲／下跌1%時，個股的漲跌預計將以 β% 的比例相應上漲或下跌。    		 台股 美(ETF)
    波動率 選擇權／權證商品標的股的歷史波動率。
    單位是％，例如數值如果是23.12％，則回傳23.12。    		 台股 選擇權
    股價淨值比 計算公式：收盤價／每股淨值。 台股 美(股票)
    振幅 當期股價的振幅。單位為%。 台股
    真實範圍 股價的真實範圍(TrueRange)，數值是考慮到跳空所推算出來的股價真正震盪幅度。 台股
    真實範圍波幅 真實範圍(TrueRange) / 昨收價 * 100%。單位為%，假如真實範圍是5，昨收價是50，則真實範圍波幅為10%，回傳值為10。 台股
    高低差 當期的最高價 － 當期的最低價。 台股
    參考價 當日交易的參考價。一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除權息後的參考價。 台股
    最低價 K棒資料的最低成交價位。 台股 香港股票 大陸股票 美(股票)
    最高價 K棒資料的最高成交價位。 台股 香港股票 大陸股票 美(股票)
    跌停價 當日的跌停價。 台股
    週平均收益率 最近5日的平均賺賠比例。

    計算公式 = 近5日漲幅 / 5，其中近5日漲幅 = (收盤價 - 5日前收盤價) / 5日前收盤價 * 100%。    		 台股
    開盤價 K棒資料的開盤價(第一個成交價位)。 台股 香港股票 大陸股票 美(股票)
    漲停價 當日的漲停價。 台股
    漲跌幅 當期股價的漲跌幅。單位是％。 台股 香港股票 大陸股票 美(股票)
    標準差 使用一年的每日漲跌幅，計算標準差數值，用來衡量股票報酬率的波動性。數值越小表示波動性越低。 台股
