    選股欄位 - 基本

    欄位名稱 欄位說明 支援商品
    公司成立日期 公司成立日期。
    欄位格式為西元年月日，例如20100101。    		 台股 美(ETF)
    公司風格 系統運用財務數值評價，將公司分為混合型、價值型、成長型。
    公司風格數值會每季重新評估。    		 台股
    公司掛牌日期 公司上櫃或是上市的日期。

    欄位格式為西元年月日，例如20100101。    		 台股
    公司類別 系統運用市值，將公司分為大型股、中型股、小型股。
    公司類別數值會每月重新評估。    		 台股
    公積配股 股票股利內的公積配股。
    (股票股利 = 盈餘配股 + 公積配股)    		 台股
    月營收 台股上市櫃公司每月公告的營業收入數值。 台股
    月營收月增率 商品當月營收較上一月營收的成長率。
    計算公式＝(當月營收－上月營收) / 上月營收*100%。
    單位為%，例如月營收月增率為10%，則回傳10。    		 台股
    月營收年增率 商品當月營收較去年同月營收的成長率。
    計算公式 ＝(當月營收－去年同期) / 去年同期*100%。
    單位為%，例如月營收年增率為10%，則回傳10。    		 台股
    投資建議評級 市場機構對個股的投資建議評等分數。

    美股股票才適用本欄位。    		 美(股票)
    股本(元) 股票的股本，換算成元。 台股
    股本(億) 股票的股本，換算成億元。 台股
    股利合計 最近一個完整股利年度的現金股利＋股票股利。 台股
    股票股利 最近一個完整股利年度的股票股利。 台股
    股票股利佔股利比重 計算公式：股票股利 / (現金股利+股票股利) *100%。 台股
    股票股利殖利率 股票股利(配股)相對於股價的比例。 台股
    盈餘配股 股票股利內的盈餘配股項目。
    (股票股利 = 盈餘配股 + 公積配股)    		 台股
    員工人數 來自年報的員工總人數。

    註：美股僅支援年頻率。    		 台股 美(股票)
    員工配股率 計算公式 : 員工分紅股數 / 已發行股數*100%。
    資料來源：台灣經濟新報。    		 台股
    財報股本(億) 公司財報(季、年)上載明的股本，換算成億元。 台股
    現金股利 現金配發的股利(股息)。 台股 美(股票) 美(特別股) 美(ETF)
    現金股利佔股利比重 計算公式：現金股利 / (現金股利+股票股利) * 100%。 台股
    現金股利殖利率 現金股利相對於股價的比例，用來衡量股票投資的回報。 台股
    累計營收 年初迄今公告的月營收累計總金額。 台股
    累計營收年增率 商品今年以來累計營收較去年同期累計營收的成長率。
    計算公式 = (今年以來累計營收 - 去年同期累計營收) / 去年同期累計營收 * 100%。
    單位為％，例如：累計營收年增率10%，則回傳10。    		 台股
    殖利率 股票的殖利率。單位是%，例如殖利率為5.6％的話，則回傳5.6。 台股 美(股票) 美(特別股) 美(ETF)
    發行張數(張) 當期的發行張數。 台股
    發行張數(萬張) 當期的發行張數。單位是萬張。 台股
    新產能預計量產日期 最近可能增加產能的預估量產日期。

    欄位格式為西元年月日，例如20221101。    		 台股
    董事長 目前董事長的姓名。 台股
    總市值(元) 股票或是指數的市值。股票市值以發行股數 ＊ 收盤價來計算，指數則是成分股的市值加總。單位為元。 台股 類股指數 美(股票)
    總市值(億) 股票或是指數的市值，換算成億元。股票市值以發行股數 ＊ 收盤價來計算，指數則是成分股的市值加總。 台股 類股指數
    總經理 目前總經理的姓名。 台股
