    內建函數 - 數學函數

    函數名稱 函數中文名稱 語法
    AbsValue 取得絕對值 取絕對值。傳回無正負號的數值
    回傳數值 = AbsValue(數值)
    ArcCosine 反餘弦函數 計算反餘弦函數，請傳入絕對值小於 1 的數字。
    角度 = ArcCosine(數值)
    ArcSine 反正弦函數 計算反正弦函數，請傳入絕對值小於 1 的數字。
    角度 = ArcSine(數值)
    ArcTangent 反正切函數 計算反正切函數
    角度 = ArcTangent(數值)
    AvgList 計算平均值 傳入數個數值，回傳這些數值的的平均值。
    回傳平均值 = AvgList(數值1, 數值2, 數值3, ..)
    Ceiling 小數無條件進位後轉成整數 把小數無條件進位後轉成整數
    回傳數值 = Ceiling(數值)
    Combination 計算集合可能的組合個數 計算從集合個數M內取出N個元素的可能組合數目
    回傳數值 = Combination(集合個數M, 欲取出的個數N)
    Cos 餘弦函數 計算角度的餘弦值
    餘弦值 = Cos(角度)
    Cosine 餘弦函數 計算角度的餘弦值
    數值 = Cosine(角度)
    CoTangent 餘切函數 計算餘切函數。
    回傳數值 = CoTangent(角度)
    ExpValue 計算自然對數的次方 計算自然對數次方運算後的數值
    回傳數值 = ExpValue(數值)
    Factorial 計算數字的階乘 計算數字的階乘結果
    回傳數值 = Factorial(數字)
    Floor 小數無條件捨去後轉成整數 把小數無條件捨去後轉成整數
    回傳數值 = Floor(數值)
    FracPortion 回傳數值的小數部分 計算數值的小數部分
    回傳數值 = FracPortion(數值)
    IntPortion 回傳數值的整數部分 計算傳入數值的整數部分的數值
    回傳數值 = IntPortion(數值)
    Log 計算自然對數 取以e為底的對數值，請傳入大於0的數字。
    回傳數值 = Log(數值)
    MaxList 計算最大值 計算多個數值內的最大值
    回傳數值 = MaxList(數值1, 數值2, 數值3, ..)
    MaxList2 計算最2大值 計算多個數值內的最2大值
    回傳數值 = MaxList2(數值1, 數值2, 數值3, ..)
    MinList 計算最小值 計算多個數值內的最小值
    回傳數值 = MinList(數值1, 數值2, 數值3, ..)
    MinList2 計算最2小值 計算多個數值內的最2小值
    回傳數值 = MinList2(數值1, 數值2, 數值3, ..)
    Mod 回傳兩數相除後的餘數 計算傳入的兩個數值相除後的餘數
    餘數 = Mod(被除數，除數)
    Neg 計算數值的負絕對值 將輸入的數值轉成負的絕對值
    回傳數值 = Neg(數值)
    NthMaxList 取第N大的數值 傳入多個數值，回傳這些數值內由大到小排名第幾的數字。
    回傳數值 = NthMaxList(排名位置, 數值1, 數值2, 數值3, ..)
    NthMinList 取第N小的數值 傳入多個數值，回傳這些數值內由小到大排名第幾的數字。
    回傳數值 = NthMinList(排名位置, 數值1, 數值2, 數值3, ..)
    Permutation 計算集合可能的排列個數 計算從集合個數M內取出N個元素的可能排列個數
    回傳數值 = Permutation(集合個數M, 欲取出的個數N)
    Pos 計算數值的正數值 將輸入的數值轉為正數。
    回傳數值 = Pos(數值)
    Power 計算數字乘冪 計算數字的乘冪數值
    回傳數值 = Power(底數, 指數)
    Random 回傳亂數值 回傳一個介於0跟傳入數值之間的隨機亂數
    隨機亂數 = Random(最大亂數的範圍)
    Round 執行小數的四捨五入運算 回傳數值四捨五入後的結果，可以指定小數位數。
    回傳數值 = Round(數值，小數位數)
    Sign 數值的正負號 回傳數值的正負號
    回傳數值 = Sign(數值)
    如果是正數，回傳1，如果是負數，回傳-1，如果是0，則回傳0
    Sin 正弦值 計算角度的正弦值
    正弦值 = Sin(角度)
    Sine 正弦值 計算角度的正弦值
    正弦值 = Sine(角度)
    Square 計算數值的平方 計算傳入數值的平方
    回傳數值 = Square(數值)
    SquareRoot 計算數值的平方根 計算傳入數值的平方根，請傳入大於0的數字。
    回傳數值 = SquareRoot(數值)
    SumList 計算加總 傳入數個數值，回傳這些數值的的加總
    回傳加總值 = SumList(數值1, 數值2, 數值3, ..)
    Tan 正切 計算角度的正切值
    正切值 = Tag(角度)
    Tangent 正切 計算角度的正切值
    正切值 = Tangent(角度)
