    10年年化報酬率 近10年年化報酬率，計算方式包含配息，以及配息再投入。 台股 美(ETF)
    1年年化報酬率 近1年年化報酬率，計算方式包含配息，以及配息再投入。
    		 台股 美(ETF)

    EPS預估值 市場機構對個股的季度EPS預估值。
    美股股票才適用本欄位。    		 美(股票)
    一年內到期長期負債 「資產負債表」中「流動負債」的「一年內到期長期負債」項目。
    金融、壽險、產險股不適用此欄位。    		 台股
    加權平均股數 「損益表」中「加權平均股數」項目。 台股 美(股票)
    外幣換算調整數 「資產負債表」股東權益項目中的「國外機構報表換算差額」。
    金融、壽險、產險、證券股才適用此欄位。    		 台股
    外銷比率 外銷占營收的比例。
    單位為％，例如：外銷比率為90%，則回傳90。    		 台股
    市研率 計算公式: 研發費用 / 總市值*100%。 台股
    市值營收比 計算公式: 市值 / 當期營業收入淨額。 台股
    平均收帳天數 財務比率表上的"平均收帳天數"欄位。 台股
    平均售貨天數 單期財務比率表上的"平均售貨天數"欄位。 台股
    本期稅後淨利 「損益表」中「歸屬母公司淨利（損）」項目。 台股 美(股票)
    未分配盈餘 「資產負債表」股東權益項目中的「未分配盈餘」。 台股
    未完工程及預付款 「資產負債表」中「非流動資產」項內的「預付設備款」項目。
    金融、壽險、產險、證券股不適用此欄位。    		 台股
    用人費用率 財務比率表上的「用人費用率」欄位。
    提供單季與年度資料。
    單位為%，例如：用人費用率為15%，則回傳15。    		 台股
    企業價值 計算公式: 總市值 ＋ 負債總額 － 現金及約當現金。 台股
    企業價值營收比 計算方式: (總市值 + 負債總額－現金及約當現金) / 營業收入淨額 *100%。 台股
    因子_QMJ安全 從六個指標(貝他因子、負債比率、股東權益報酬率的波動度、特質波動度、Altman’s Z分數、以及Ohlson's O分數)個別找出每項指標中某檔股票在整體市場的排名，將這六個排名的標準化數值平均，即為該檔股票的綜合安全評分。 台股
    因子_QMJ獲利 從六個指標(應計項目佔資產比率、自由現金流量佔總資產比率、毛利佔資產比率、毛利佔營收比率、資產報酬率、股東權益報酬率)個別找出每項指標中某檔股票在整體市場的排名，將這六個排名的標準化數值平均，即為該檔股票的綜合獲利能力評分。 台股
    因子_一年動能 股票在過去1年內(不包含當月，共計11個月)的漲跌幅度。 台股
    因子_日週轉率流動性 過去一個月的日週轉率平均值。 台股
    因子_月週轉率流動性 自然對數(標準差(股票過去36個月的月週轉率))。 台股
    因子_半年動能因子 股票在過去1年至過去半年這段期間內(12個月前至7個月前)的漲跌幅度。 台股
    因子_成交值流動性 自然對數(過去一個月成交金額加總)。 台股
    因子_貝他值 過去一年的日報酬率和市場報酬率之共變異數/市場報酬率之變異數。 台股
    因子_股票發行量 自然對數(當期股票面額/上季股票面額/前一季流通在外股數)。 台股
    因子_前一年獲利 一年前的營業利益/三年前的帳面權益。 台股
    因子_前三年獲利 三年前的營業利益/三年前的帳面權益。 台股
    因子_前兩年獲利 兩年前的營業利益/三年前的帳面權益。 台股
    因子_特質波動度 透過對歷史股票報酬與市場報酬進行Fama French三因子統計迴歸分析，並計算迴歸殘差值的標準差，即為特質波動度因子。 台股
    因子_偏態係數 日報酬率序列的三階主動差/日報酬率序列的二階主動差的1.5次方根。 台股
    因子_帳面市值比 自然對數(帳面價值/市場價值)。 台股
    因子_彩券型需求 過去一個月中最大五筆日報酬率的簡單算數平均。 台股
    因子_淨營運資產 淨營運資產/去年同期總資產。 台股
    因子_現金流量波動度 標準差(過去16季之營業現金流量/營業收入)。 台股
    因子_最高價動能 當前收盤價與過去52週內最高價的比值。 台股
    因子_實現損益偏態係數 偏態係數分子/偏態係數分母。
    偏態係數分子=過去一個月對數日報酬率的平方加總，再開3/2次方根。
    偏態係數分母=過去一個月對數日報酬率的三次方加總，並乘以上個月交易天數的平方根。    		 台股
    因子_潛在上檔報酬比 將過去1年的日報酬率隨機抽樣1000筆，以正報酬率的加總與過去1年日交易天數的比值作為分子，以負報酬率的平方加總與過去1年日交易天數的比值平方根作為分母，得到的比率即為潛在上檔報酬比率因子。 台股
    因子分數_QMJ安全 QMJ安全因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_QMJ獲利 QMJ獲利因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_一年動能 動能因子(以前11個月報酬為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_日週轉率流動性 流動性因子(以平均日週轉率為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_月週轉率流動性 流動性因子(以股票月週轉率標準差為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_半年動能因子 動能因子(以前7~12月報酬為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_成交值流動性 流動性因子(以成交值為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_貝他值 貝他因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_股東權益報酬率 股東權益報酬率因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_股票發行量 股票發行量因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_前一年獲利 獲利因子(以前一年獲利為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_前三年獲利 獲利因子(以前三年獲利為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_前兩年獲利 獲利因子(以前兩年獲利為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_特質波動度 特質波動度因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_偏態係數 偏態係數因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_帳面市值比 帳面市值比因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_彩券型需求 彩券型需求因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_淨營運資產 淨營運資產因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_現金流量波動度 現金流量波動度因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_現金殖利率 現金殖利率因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_最高價動能 動能因子(以前52週高價為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_實現損益偏態係數 偏態係數因子(以實現損益為基準)經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    因子分數_潛在上檔報酬比 潛在上檔報酬比率因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名分數。 台股
    存貨 「資產負債表」中「流動資產」項內的「存貨」項目。

    存貨週轉率(次) 財務比率表上的「存貨週轉率(次)」欄位。

    存貨營收比 計算公式: 存貨之商品及製成品 / 營業收入淨額*100%。 台股
    年報酬率 僅支援台/美ETF商品.

    自由現金流量 現金流量表中的「"來自營運之現金流量" + "購置不動產廠房設備（含預付）－CFI" + "支付利息－CFF"」。

    		 台股

    		 台股 美(股票)
    投資跌價損失 「損益表」中「金融資產減損損失」項目。 台股
    投資跌價損失回轉 「損益表」中「金融資產減損迴轉利益」項目。 台股
    杜邦型ROA 計算公式: 總資產週轉率(次) * 稅後淨利率。 台股
    杜邦型ROE 計算公式: 總資產週轉率(次) * 稅後淨利率 * (資產總額 / 股東權益總額)。 台股
    每人營業利益 財務比率表上的「每人營業利益」欄位。
    提供單季與年度資料。
    不提供此欄位的公司，輸出值會以零表示。
    		 台股
    每股自由現金流量 計算公式：自由現金流量／加權平均股數 台股
    每股流動淨資產 計算公式: (流動資產－全部負債) ／ 普通股股數。 台股

    提供單季與年度資料。    		 台股 美(股票)

    提供單季與年度資料。    		 台股

    提供單季與年度資料。    		 台股
