隱含波動率是經由選擇權市場價格反推而得的波動率。隱含波動率的水準顯示了選擇權價格的高低（貴或便宜），也代表了市場參與者對未來的預期。通常會搭配歷史波動率來判斷市場是否出現狂熱或恐慌的狀況。

範例：

```
value1 = IVolatility("C",8800,9000,20,2,0,40);       //計算8800買權的隱含波動率
plot1(value1,"隱含波動率");
```

```
value1 = IVolatility("C",8800,9000,20,2,0,40);       //計算8800買權的隱含波動率
plot1(value1,"隱含波動率");
```
