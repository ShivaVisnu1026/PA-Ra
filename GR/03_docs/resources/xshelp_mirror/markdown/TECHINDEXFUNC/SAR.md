由威爾德（Wilder）發明的停損點轉向操作系統（SAR, Stop And Reverse）或者稱之為拋物線型指標（Parabolic Time/Price），是一種隨著時間的延續，用類似於拋物曲線的追趕方式，來追趕價位行情，以便設定出買賣操作時的停損點反轉值，使得一旦行情觸及停損反轉時，可以依照訊號進行買賣的動作。

範例：

```
value1 = SAR(0.02, 0.02, 0.2);       //計算SAR指標
plot1(value1, "SAR");
```

```
value1 = SAR(0.02, 0.02, 0.2);       //計算SAR指標

```
