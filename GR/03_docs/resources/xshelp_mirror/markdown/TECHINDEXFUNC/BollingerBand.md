包寧傑通道為John Bollinger在1980年代發明的指標。以價格均線為中心點，往上N個標準差為通道上限（壓力線）；往下減去N個標準差，為通道下限（支撐線）。

範例：

```
value1 = BollingerBand(Close,20,2);       //計算20期、2個標準差寬的包寧傑通道上限
plot1(value1, "BollingerUpperBand");
```

```
value1 = BollingerBand(Close,20,2);       //計算20期、2個標準差寬的包寧傑通道上限
plot1(value1, "BollingerUpperBand");
```
