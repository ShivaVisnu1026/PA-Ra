```
Value1 = GetQuote("營收年增率");
Value1 = GetQuote("RevenueYoY");
Value1 = GetQuote("營收年增率%");
Value1 = q_RevenueYoY;
```

最新一期營收較去年同期營收的成長率。單位為%，例如營收年增率為10%，則回傳10。

計算公式 ＝(本期營收－去年同期) / 去年同期*100%。

台股營收是每月一期，陸股/美股是每季一期，而港股則是每半年一期。
