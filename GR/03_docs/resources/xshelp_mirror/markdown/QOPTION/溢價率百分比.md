損益兩平價格與標的價格的差異幅度比例。數值越大，代表要達到損益兩平越不容易，風險就越大。

認購權證的溢價率為(損益兩平 ─ 標的股價格)／標的股價格＊100％。
認售權證的溢價率為(標的股價格 ─ 損益兩平)／標的股價格＊100％。

舉例而言，如果一檔認購權證的損益兩平價格是105，而標的股的價格是90，那麼溢價率＝(105-90)/90 * 100% = 16.67%，欄位回傳值則為16.67。
