理論價格與市場賣價差佔賣價的比例。
計算公式＝(理論價 ─ 賣價)／賣價 ＊ 100%。

假設權證的理論價格是1.5，市場最佳一檔賣價是1.6，那麼獲利率為(1.5 - 1.6) / 1.6 * 100 = -6.25%，欄位的回傳值則為-6.25。
