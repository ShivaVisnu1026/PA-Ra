範例：

```
Array: arrA[5](0); // 宣告arrA是一個有5個元素的陣列，初始值都是0


arrA[1] = 0;  arrA[2] = 10; arrA[3] = 20; arrA[4] = 30; arrA[5] = 40;


value1 = NthLowestArray(arrA,5,3); //計算陣列arrA的第三個極小值
plot1(value1);                      //繪製陣列arrA的第三個極小值的連線
```
