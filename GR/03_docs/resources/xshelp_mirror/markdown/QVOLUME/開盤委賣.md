開盤時委託賣出的數量。也可以使用 GetField("開盤委賣", "D") 來表示。

台股的數值是開盤第一筆成交前五檔委賣的數量加總。

類股商品（細產業指標、主題指標）的計算方式則是成分股的開盤委賣的加總。

大盤/期貨/選擇權的數值則是交易所公布的委買委賣資料的第一筆數值。大盤的數值會扣除權證。
