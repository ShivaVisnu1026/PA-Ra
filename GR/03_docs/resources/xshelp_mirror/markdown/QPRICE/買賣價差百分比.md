買賣價差占買價的比例。
計算公式：(賣價 ─ 買價)／買價＊100%。

例如：最佳一檔的買進價位為14.25，最佳一檔賣出價位為14.40，則買賣價差比為(14.40 - 14.25) /14.25*100%=1.05%，回傳值為1.05。

當買賣價差百分比越大時，表示要成交的話必需支付更多的成本(要花更多的錢去追逐賣價)。
