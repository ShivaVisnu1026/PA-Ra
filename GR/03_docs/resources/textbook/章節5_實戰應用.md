## 💻 程式碼範例

### 基本自動交易腳本
```xs
{@type:autotrade}
// 自動交易腳本：移動平均交叉策略
input: ma_short(5);           // 短期均線期數
input: ma_long(20);           // 長期均線期數
input: stop_loss(0.02);       // 停損比例
input: take_profit(0.04);     // 停利比例

// 計算均線
value1 = Average(Close, ma_short);
value2 = Average(Close, ma_long);

// 進場條件：金叉
if value1 crosses above value2 then
    SetPosition(1, Close);    // 買進 1 口

// 進場條件：死叉
if value1 crosses under value2 then
    SetPosition(-1, Close);   // 賣出 1 口

// 停損停利
if Position > 0 then
begin
    if Close < FilledAvgPrice * (1 - stop_loss) then
        SetPosition(0, Close);  // 停損
    if Close > FilledAvgPrice * (1 + take_profit) then
        SetPosition(0, Close);  // 停利
end;

if Position < 0 then
begin
    if Close > FilledAvgPrice * (1 + stop_loss) then
        SetPosition(0, Close);  // 停損
    if Close < FilledAvgPrice * (1 - take_profit) then
        SetPosition(0, Close);  // 停利
end;
```

### 進階交易策略
```xs
{@type:autotrade}
// 進階策略：RSI 均值回歸策略
input: rsi_length(14);        // RSI 計算期數
input: rsi_oversold(30);      // RSI 超賣線
input: rsi_overbought(70);    // RSI 超買線
input: position_size(1);      // 部位大小

// 計算 RSI
value1 = RSI(Close, rsi_length);

// 進場條件：RSI 超賣
if value1 < rsi_oversold and Position = 0 then
    SetPosition(position_size, Close);

// 進場條件：RSI 超買
if value1 > rsi_overbought and Position = 0 then
    SetPosition(-position_size, Close);

// 出場條件：RSI 回到中性區域
if Position > 0 and value1 > 50 then
    SetPosition(0, Close);
if Position < 0 and value1 < 50 then
    SetPosition(0, Close);
```

### 風險控制機制
```xs
{@type:autotrade}
// 風險控制：動態停損策略
input: initial_stop(0.02);    // 初始停損比例
input: trail_stop(0.01);      // 追蹤停損比例
input: max_position(2);       // 最大部位

variable: highest_price(0);   // 最高價記錄
variable: lowest_price(0);    // 最低價記錄

// 部位控制
if AbsValue(Position) > max_position then
    SetPosition(Sign(Position) * max_position, Close);

// 多單風險控制
if Position > 0 then
begin
    // 更新最高價
    if Close > highest_price then
        highest_price = Close;
    
    // 追蹤停損
    if Close < highest_price * (1 - trail_stop) then
    begin
        SetPosition(0, Close);
        highest_price = 0;
    end;
end;

// 空單風險控制
if Position < 0 then
begin
    // 更新最低價
    if Close < lowest_price or lowest_price = 0 then
        lowest_price = Close;
    
    // 追蹤停損
    if Close > lowest_price * (1 + trail_stop) then
    begin
        SetPosition(0, Close);
        lowest_price = 0;
    end;
end;
```

### 日內交易策略
```xs
{@type:autotrade}
// 日內交易：開盤突破策略
input: break_threshold(0.01); // 突破閾值
input: volume_ratio(2.0);     // 成交量倍數

// 計算開盤價和成交量
value1 = Open;                // 開盤價
value2 = Average(Volume, 20); // 平均成交量

// 突破條件
condition1 = Close > value1 * (1 + break_threshold);  // 向上突破
condition2 = Close < value1 * (1 - break_threshold);  // 向下突破
condition3 = Volume > value2 * volume_ratio;          // 成交量放大

// 進場條件
if condition1 and condition3 and Position = 0 then
    SetPosition(1, Close);    // 買進

if condition2 and condition3 and Position = 0 then
    SetPosition(-1, Close);   // 賣出

// 收盤前平倉
if Time >= 1330 then          // 13:30 後平倉
    SetPosition(0, Close);
```

## 🔧 重要語法

### 自動交易語法
- `SetPosition(數量, 價格);` - 設定交易部位
- `Position` - 當前部位
- `FilledAvgPrice` - 成交均價
- `Filled` - 成交數量

### 交易指令語法
- `MARKET` - 市價單
- `AddSpread(價格, 檔數)` - 檔位換算
- `CancelAllOrders` - 取消所有委託

### 風險控制語法
- `SetPosition(0, 價格);` - 平倉
- `Position > 0` - 多單部位
- `Position < 0` - 空單部位
