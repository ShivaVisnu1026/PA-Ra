# 章節 4：XScript 核心功能

## 📚 章節概述
本章節介紹 XScript 的核心功能，包括選股腳本、警示腳本、指標腳本的撰寫方法，以及各種腳本類型的特點和應用場景。

## 📖 主要內容

### 選股腳本 (Filter)
- 基本選股條件設定
- 複合條件組合
- 選股結果輸出
- 選股腳本最佳實踐

### 警示腳本 (Sensor)
- 警示條件設定
- 警示觸發機制
- 警示腳本優化
- 即時警示應用

### 指標腳本 (Indicator)
- 技術指標計算
- 指標圖表顯示
- 指標參數設定
- 自訂指標開發

### 函數腳本 (Function)
- 自訂函數開發
- 函數參數設計
- 函數回傳值處理
- 函數重複使用

## 💻 程式碼範例

### 選股腳本範例
```xs
{@type:filter}
// 選股腳本：找出符合多項條件的股票
input: ma_short(5);           // 短期均線期數
input: ma_long(20);           // 長期均線期數
input: rsi_length(14);        // RSI 計算期數
input: volume_ratio(1.5);     // 成交量倍數

// 計算技術指標
value1 = Average(Close, ma_short);    // 短期均線
value2 = Average(Close, ma_long);     // 長期均線
value3 = RSI(Close, rsi_length);      // RSI 指標
value4 = Average(Volume, 20);         // 平均成交量

// 選股條件
condition1 = Close > value1;          // 價格突破短期均線
condition2 = value1 > value2;         // 短期均線在長期均線之上
condition3 = value3 > 50;             // RSI 大於 50
condition4 = Volume > value4 * volume_ratio; // 成交量放大

// 綜合條件
if condition1 and condition2 and condition3 and condition4 then
    ret = 1;

// 輸出欄位
SetOutputName1("短期均線");
OutputField1(value1);
SetOutputName2("RSI");
OutputField2(value3);
```

### 警示腳本範例
```xs
{@type:sensor}
// 警示腳本：股價突破重要均線時發出警示
input: ma_length(20);         // 均線期數
input: volume_threshold(2.0); // 成交量閾值

// 計算均線和成交量
value1 = Average(Close, ma_length);
value2 = Average(Volume, 20);

// 突破條件
condition1 = Close > value1;                    // 價格突破均線
condition2 = Close[1] <= value1[1];             // 前一日未突破
condition3 = Volume > value2 * volume_threshold; // 成交量放大

// 觸發警示
if condition1 and condition2 and condition3 then
    ret = 1;

// 設定所需 K 棒數量
SetTotalBar(ma_length + 5);
```

### 指標腳本範例
```xs
{@type:indicator}
// 自訂指標：多空力道指標
input: length(14);            // 計算期數

// 計算多空力道
variable: bull_power(0);      // 多方力道
variable: bear_power(0);      // 空方力道

bull_power = High - Average(Close, length);
bear_power = Low - Average(Close, length);

// 輸出指標
SetOutputName1("多方力道");
OutputField1(bull_power);
SetOutputName2("空方力道");
OutputField2(bear_power);

// 設定圖表顯示
SetPlotColor(1, Red);
SetPlotColor(2, Blue);
```

### 函數腳本範例
```xs
{@type:function}
// 自訂函數：計算波動率
input: price(numericseries), length(numericsimple);

variable: sum_squared(0);
variable: i(0);

// 計算標準差
sum_squared = 0;
for i = 1 to length do
begin
    sum_squared = sum_squared + (price[i] - Average(price, length)) * (price[i] - Average(price, length));
end;

// 回傳波動率
Volatility = Sqrt(sum_squared / length);
```

## 🔧 重要語法

### 腳本類型宣告
- `{@type:filter}` - 選股腳本
- `{@type:sensor}` - 警示腳本
- `{@type:indicator}` - 指標腳本
- `{@type:function}` - 函數腳本

### 選股腳本語法
- `ret = 1;` - 符合選股條件
- `ret = 0;` - 不符合選股條件
- `OutputField1(值);` - 輸出欄位

### 警示腳本語法
- `ret = 1;` - 觸發警示
- `ret = 0;` - 不觸發警示
- `SetTotalBar(數量);` - 設定所需 K 棒數量

### 指標腳本語法
- `OutputField1(值);` - 輸出指標值
- `SetOutputName1("名稱");` - 設定輸出名稱
- `SetPlotColor(1, 顏色);` - 設定圖表顏色
