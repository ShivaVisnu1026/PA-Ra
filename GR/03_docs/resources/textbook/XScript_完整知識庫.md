## 💻 程式碼範例庫

### 基礎語法範例
```xs
// 變數宣告
input: length(20);
variable: ma_value(0);

// 條件判斷
if Close > Average(Close, length) then
    ret = 1;

// 迴圈結構
for i = 1 to 10 do
begin
    value1 = value1 + Close[i];
end;
```

### 函數使用範例
```xs
// 內建函數
value1 = RSI(Close, 14);
value2 = MACD(Close, 12, 26, 9);

// 自訂函數
function: CustomMA(price, length)
begin
    CustomMA = Average(price, length);
end;
```

### 選股腳本範例
```xs
{@type:filter}
input: ma_length(20);
input: rsi_length(14);

value1 = Average(Close, ma_length);
value2 = RSI(Close, rsi_length);

if Close > value1 and value2 > 50 then
    ret = 1;
```

### 警示腳本範例
```xs
{@type:sensor}
input: threshold(0.02);

if Close > Close[1] * (1 + threshold) then
    ret = 1;
```

### 自動交易範例
```xs
{@type:autotrade}
input: ma_short(5);
input: ma_long(20);

value1 = Average(Close, ma_short);
value2 = Average(Close, ma_long);

if value1 crosses above value2 then
    SetPosition(1, Close);
```

## 🔧 重要語法參考

### 腳本類型
- `{@type:filter}` - 選股腳本
- `{@type:sensor}` - 警示腳本
- `{@type:indicator}` - 指標腳本
- `{@type:function}` - 函數腳本
- `{@type:autotrade}` - 自動交易腳本

### 變數宣告
- `input: 變數名(預設值);` - 輸入參數
- `variable: 變數名(初始值);` - 一般變數
- `value1` 到 `value99` - 系統變數

### 控制結構
- `if 條件 then 動作;` - 條件判斷
- `for 變數 = 起始值 to 結束值 do 動作;` - for 迴圈
- `while 條件 do 動作;` - while 迴圈

### 函數呼叫
- `函數名(參數1, 參數2, ...)` - 基本呼叫
- `函數名[偏移量]` - 歷史值
- `函數名(參數)[偏移量]` - 帶參數歷史值
