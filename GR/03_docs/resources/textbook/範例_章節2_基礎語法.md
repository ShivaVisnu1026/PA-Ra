# 章節 2：XScript 基礎語法

## 📚 章節概述
本章節介紹 XScript 的基本語法規則，包括變數宣告、資料型別、運算子等基礎概念，是學習 XScript 的重要基礎。

## 📖 主要內容

### 變數宣告
XScript 支援多種變數宣告方式：
- **input**：輸入參數，可在執行時調整
- **variable**：一般變數，用於儲存計算結果
- **value1-value99**：系統預設變數，可直接使用

### 資料型別
- **數值型別**：整數、浮點數
- **字串型別**：文字資料
- **布林型別**：true/false
- **時間序列**：OHLCV 資料

### 運算子
- **算術運算子**：+、-、*、/、%
- **比較運算子**：>、<、>=、<=、=、<>
- **邏輯運算子**：and、or、not

## 💻 程式碼範例

### 基本變數宣告
```xs
// 輸入參數宣告
input: length(20);           // 移動平均期數
input: threshold(0.5);       // 閾值參數

// 一般變數宣告
variable: ma_value(0);       // 移動平均值
variable: signal(0);         // 訊號值

// 使用系統變數
value1 = Close;              // 收盤價
value2 = Volume;             // 成交量
```

### 條件判斷
```xs
// 基本條件判斷
if Close > Open then
    ret = 1;                 // 上漲時觸發

// 複合條件判斷
if Close > Average(Close, 20) and Volume > Average(Volume, 10) then
    ret = 1;                 // 價格和成交量都超過平均時觸發

// 多條件判斷
if condition1 and condition2 then
    ret = 1
else if condition3 then
    ret = 0
else
    ret = -1;
```

### 迴圈結構
```xs
// for 迴圈範例
variable: sum(0);
for i = 1 to 10 do
begin
    sum = sum + Close[i];
end;
value1 = sum / 10;           // 計算 10 期平均

// while 迴圈範例
variable: count(0);
while count < 5 do
begin
    count = count + 1;
    // 執行某些操作
end;
```

## 🔧 重要語法

### 變數宣告語法
- `input: 變數名(預設值);` - 宣告輸入參數
- `variable: 變數名(初始值);` - 宣告一般變數
- `value1` 到 `value99` - 系統預設變數

### 條件判斷語法
- `if 條件 then 動作;` - 基本條件判斷
- `if 條件 then 動作1 else 動作2;` - 完整條件判斷
- `if 條件 then begin ... end;` - 多行動作

### 迴圈語法
- `for 變數 = 起始值 to 結束值 do 動作;` - for 迴圈
- `while 條件 do 動作;` - while 迴圈

## 💡 實用技巧

### 變數命名技巧
- 使用有意義的變數名稱
- 避免使用保留字
- 保持命名一致性

### 程式碼組織技巧
- 適當使用註解說明
- 保持一致的縮排格式
- 將相關邏輯分組

### 效能優化技巧
- 避免不必要的計算
- 合理使用 SetTotalBar
- 優化迴圈結構

## ⚠️ 注意事項

### 語法注意事項
- 每行結尾必須加分號 `;`
- 大小寫敏感
- 保留字不能作為變數名

### 效能注意事項
- 避免在迴圈中進行複雜計算
- 合理設定 SetTotalBar 參數
- 注意記憶體使用

### 除錯注意事項
- 使用 OutputField 輸出除錯資訊
- 檢查變數初始值
- 注意陣列邊界
