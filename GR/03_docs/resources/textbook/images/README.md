# XScript 教科書圖檔資料夾

本資料夾存放從 PDF 教科書轉換而來的 PNG 圖檔。

## 📁 檔案命名建議

請按照以下格式命名圖檔：

### 章節結構
```
01_基礎語法_001.png
01_基礎語法_002.png
01_基礎語法_003.png
02_函數使用_001.png
02_函數使用_002.png
03_選股腳本_001.png
03_選股腳本_002.png
04_警示腳本_001.png
04_警示腳本_002.png
05_自動交易_001.png
05_自動交易_002.png
```

### 或者簡單編號
```
page_001.png
page_002.png
page_003.png
...
```

## 🔍 處理流程

1. **放置圖檔**：將所有 PNG 圖檔放到此資料夾
2. **AI 讀取**：AI 會逐一讀取圖檔內容
3. **轉換為文字**：將圖檔內容轉換為 Markdown 格式
4. **建立知識庫**：整理成結構化的學習資料

## 📖 輸出檔案

AI 會建立以下檔案：
- `XScript_教科書_完整版.md` - 完整的教科書內容
- `XScript_基礎語法.md` - 基礎語法章節
- `XScript_函數參考.md` - 函數使用章節
- `XScript_範例集.md` - 程式範例集合
- `XScript_最佳實踐.md` - 最佳實踐和技巧

## ⚠️ 注意事項

- 圖檔解析度建議至少 300 DPI
- 檔案大小建議不超過 10MB 每張
- 確保文字清晰可讀
- 按照章節順序命名檔案

## 🎯 學習效果

完成後，AI 將能夠：
- 掌握教科書中的所有 XScript 語法
- 使用教科書中的最佳實踐
- 參考教科書的程式範例
- 提供更專業的 XScript 建議和程式撰寫
