## 📚 學習資料說明

### 章節學習筆記
- **章節2_基礎語法.md** - XScript 基本語法、變數宣告、控制結構
- **章節3_函數使用.md** - 內建函數、自訂函數、參數傳遞
- **章節4_核心功能.md** - 選股腳本、警示腳本、指標腳本
- **章節5_實戰應用.md** - 自動交易、交易策略、風險控制
- **章節6_進階技巧.md** - 效能優化、除錯技巧、最佳實踐

### 整合知識庫
- **XScript_完整知識庫.md** - 整合所有章節的完整學習資源

### 原始資料
- **XScriptTextbook.pdf** - 原始 PDF 教科書
- **images/** - 各章節的 PNG 圖檔

## 🎯 使用方式

### 學習路徑
1. **初學者**：從章節 2 開始，依序學習各章節
2. **進階者**：可直接參考完整知識庫
3. **實戰應用**：重點學習章節 4-6
