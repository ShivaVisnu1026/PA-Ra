## 💻 程式碼範例

### 內建函數使用
```xs
// 價格函數
value1 = Close;                    // 收盤價
value2 = High;                     // 最高價
value3 = Low;                      // 最低價
value4 = Open;                     // 開盤價
value5 = Volume;                   // 成交量

// 技術指標函數
value6 = Average(Close, 20);       // 20 期移動平均
value7 = RSI(Close, 14);           // 14 期 RSI
value8 = MACD(Close, 12, 26, 9);   // MACD 指標

// 統計函數
value9 = Sum(Close, 10);           // 10 期收盤價總和
value10 = Max(High, 5);            // 5 期最高價最大值
value11 = Min(Low, 5);             // 5 期最低價最小值
```

### 自訂函數定義
```xs
// 自訂函數：計算乖離率
function: BiasRatio(price, length)
begin
    variable: ma_value(0);
    ma_value = Average(price, length);
    if ma_value <> 0 then
        BiasRatio = (price - ma_value) / ma_value * 100
    else
        BiasRatio = 0;
end;

// 使用自訂函數
value1 = BiasRatio(Close, 20);     // 計算 20 期乖離率
```

### 函數在選股腳本中的應用
```xs
{@type:filter}
// 選股腳本：找出 RSI 超賣且價格突破均線的股票
input: rsi_length(14);
input: ma_length(20);
input: rsi_oversold(30);

value1 = RSI(Close, rsi_length);   // 計算 RSI
value2 = Average(Close, ma_length); // 計算移動平均

// 選股條件
if value1 < rsi_oversold and Close > value2 then
    ret = 1;
```

### 函數在警示腳本中的應用
```xs
{@type:sensor}
// 警示腳本：MACD 金叉警示
input: fast_length(12);
input: slow_length(26);
input: signal_length(9);

value1 = MACD(Close, fast_length, slow_length, signal_length);
value2 = MACD_Signal(Close, fast_length, slow_length, signal_length);

// 金叉條件
if value1 crosses above value2 then
    ret = 1;
```

## 🔧 重要語法

### 函數呼叫語法
- `函數名(參數1, 參數2, ...)` - 基本函數呼叫
- `函數名[偏移量]` - 取得歷史值
- `函數名(參數)[偏移量]` - 帶參數的歷史值

### 自訂函數定義語法
```xs
function: 函數名(參數1, 參數2, ...)
begin
    // 函數內容
    函數名 = 回傳值;
end;
```

### 參數傳遞語法
- **數值參數**：直接傳遞數值
- **字串參數**：用引號包圍字串
- **布林參數**：true/false

### 函數使用注意事項
- 確保函數參數的型別正確

- 函數名稱不能與內建函數衝突
- 確保函數有正確的回傳值
- 注意函數的參數驗證

- 複雜函數會影響執行效能
- 合理使用函數的快取機制
- 避免不必要的函數呼叫
