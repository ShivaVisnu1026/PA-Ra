# 波動率指標使用指南

## 概述

本指南說明如何使用波動率指標來計算股票或指數的平均每日波動範圍和上漲/下跌波動率。

## 指標說明

### 1. ADR (Average Daily Range) - 平均每日波動範圍

**公式：**
```
ADR = Average(High - Low) over N days
```

**計算方式：**
- 取每日的波動範圍（最高價 - 最低價）
- 計算過去 N 日的平均值
- 可表示為絕對值（$）或百分比（%）

**使用時機：**
- 判斷當日波動是否高於或低於平均
- 設定停損停利目標
- 評估股票的正常波動範圍

**範例：**
如果 AAPL 的 20 日 ADR 為 $3.50（2.3%）：
- 當日範圍 $5.00 → 高於平均（1.43x ADR）
- 當日範圍 $2.00 → 低於平均（0.57x ADR）

### 2. ATR (Average True Range) - 平均真實波動範圍

**公式：**
```
True Range = Max of:
  - Current High - Current Low
  - |Current High - Previous Close|
  - |Current Low - Previous Close|

ATR = Exponential Moving Average of True Range (typically 14 periods)
```

**為什麼使用 ATR？**
- 考慮交易日之間的跳空缺口
- 對於經常跳空的股票更準確
- 使用指數移動平均平滑處理

**使用時機：**
- 類似 ADR，但更適合有跳空的股票
- 用於停損停利設定
- 用於部位大小計算

### 3. 平均上漲/下跌波動率

**公式：**
```
平均上漲波動率 = Average((High - Open) / Open * 100) over N days
平均下跌波動率 = Average((Open - Low) / Open * 100) over N days
```

**這直接回答您的問題！**

如果您想知道「過去 20 日的平均上漲或下跌波動率是 +3% 或 -3%」：

1. **範圍基礎方法：**
   - 計算每日 (High - Open) / Open * 100
   - 平均這些值 → 平均上漲 %
   - 計算每日 (Open - Low) / Open * 100
   - 平均這些值 → 平均下跌 %

2. **淨波動方法：**
   - 計算每日 (Close - Open) / Open * 100
   - 平均正數日 → 平均上漲日的漲幅
   - 平均負數日 → 平均下跌日的跌幅

**範例輸出：**
```
平均上漲波動率: +3.2% ($4.50)
平均下跌波動率: -2.8% ($3.90)
```

這告訴您：
- 平均而言，股票從開盤到高點上漲 3.2%
- 平均而言，股票從開盤到低點下跌 2.8%
- 股票在上漲方向略為更波動

## 常用週期

不同時間框架用於不同目的：

- **14 日**：標準 ATR 週期，捕捉近期波動
- **20 日**：常見 ADR 週期，平衡近期與歷史
- **30 日**：較長期平均，更平滑但反應較慢
- **60 日**：長期平均，非常平滑但可能錯過近期變化

## 實務應用

### 設定停利目標

**方法 1：ADR 基礎目標**
```
目標價 = 進場價 ± (ADR × 倍數)
```
- 保守：0.5x ADR
- 適中：1x ADR
- 積極：2x ADR

**方法 2：上漲/下跌波動率**
```
做多目標 = 進場價 × (1 + 平均上漲 %)
做空目標 = 進場價 × (1 - 平均下跌 %)
```

### 設定停損

**方法 1：ADR 基礎停損**
```
停損 = 進場價 ± (ADR × 0.5)
```

**方法 2：ATR 基礎停損**
```
停損 = 進場價 ± (ATR × 1.5)
```

**方法 3：百分比基礎**
```
停損 = 進場價 × (1 - 平均下跌 % × 0.5)
```

### 部位大小計算

**ATR 基礎部位大小：**
```
部位大小 = (帳戶風險 %) / (ATR / 價格)
```

範例：
- 帳戶：$10,000
- 每筆交易風險：1% = $100
- 股票價格：$150
- ATR：$3.00
- ATR %：2%
- 部位大小：$100 / 0.02 = $5,000 的股票

## 解讀指標

### ADR 比率

- **> 1.2**：當日波動高於平均（異常波動）
- **0.8 - 1.2**：正常波動範圍
- **< 0.8**：當日波動低於平均（可能醞釀突破）

### 上漲/下跌波動率

- **上漲 > 下跌**：股票在上漲方向更波動
- **下跌 > 上漲**：股票在下跌方向更波動
- **兩者接近**：雙向波動平衡

### 實務建議

1. **使用多個週期**：不要只依賴一個時間框架
2. **與市場比較**：這個股票的波動率相對於市場正常嗎？
3. **定期更新**：波動率會變化，每週重新計算
4. **考慮背景**：財報、新聞、事件會影響波動率
5. **結合其他指標**：將波動率與趨勢、成交量等結合使用

## 常見問題

### Q: 應該使用 ADR 還是 ATR？

**A:** 
- 如果股票經常跳空，使用 ATR
- 如果跳空較少，ADR 即可
- 兩者都計算並比較，選擇更適合的

### Q: 如何知道當日的波動是否異常？

**A:** 
查看 ADR 比率：
- 如果 > 1.5，可能是異常高波動
- 如果 < 0.5，可能是異常低波動（可能醞釀突破）

### Q: 上漲/下跌波動率如何用於交易？

**A:**
- **做多時**：預期潛在漲幅可達平均上漲 %
- **做空時**：預期潛在跌幅可達平均下跌 %
- **停損設定**：使用平均下跌 % 的 0.5-1 倍
- **停利設定**：使用平均上漲 % 的 0.75-1.5 倍
