## 🎯 學習路徑

### **初學者**
1. 閱讀 [統一學習指南](../統一學習指南.md)
2. 學習基礎語法
3. 練習簡單的警示腳本

### **進階者**
1. 學習進階語法技巧
2. 開發複雜的交易策略
3. 掌握 Group 語法應用

### **專家**
1. 研究官方範例庫
2. 開發自訂指標
3. 優化策略性能

1. **循序漸進** - 從基礎開始，逐步學習進階內容
2. **實戰導向** - 每個概念都要有實際應用
3. **持續練習** - 通過不斷實作加深理解
4. **記錄心得** - 建立自己的學習筆記
