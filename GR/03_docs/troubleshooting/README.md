## 🚨 緊急問題排除

### **腳本無法編譯**
1. 檢查保留字使用
2. 檢查語法錯誤
3. 檢查函數使用

### **腳本無法執行**
1. 檢查週期設定
2. 檢查資料讀取
3. 檢查邏輯錯誤

### **結果不如預期**
1. 檢查條件邏輯
2. 檢查變數計算
3. 檢查時間控制

## 📋 除錯步驟

### **1. 檢查編譯錯誤**
- 查看錯誤訊息
- 檢查語法規則
- 修正保留字問題

### **2. 檢查邏輯錯誤**
- 逐步檢查條件
- 驗證變數計算
- 測試邊界情況

### **3. 檢查實戰問題**
- 分析交易邏輯
- 檢查風險控制
- 優化策略參數

## 🎯 預防措施

### **編譯前檢查**
- 使用正確的語法
- 避免保留字
- 檢查函數參數

### **執行前檢查**
- 驗證週期設定
- 檢查資料完整性
- 測試基本邏輯

### **部署前檢查**
- 完整測試策略
- 檢查風險控制
- 驗證結果正確性
