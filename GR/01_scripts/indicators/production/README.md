# 波動率指標腳本

## 概述

本資料夾包含用於計算平均每日波動範圍（ADR）、平均真實波動範圍（ATR）以及上漲/下跌波動率的指標腳本。

## 可用指標

### 1. Volatility_Indicators.xs
**完整波動率指標**
- 功能：計算並顯示所有波動率指標
- 包含：ADR、ATR、上漲/下跌波動率
- 適用：指標圖表（建議日線）
- 參數：
  - `ADR_Period` (20): ADR 計算週期
  - `ATR_Period` (14): ATR 計算週期
  - `UpsideDownside_Period` (20): 上漲/下跌波動率計算週期

### 2. ADR_Indicator.xs
**ADR 專用指標**
- 功能：專注於平均每日波動範圍計算
- 顯示：ADR 絕對值、百分比、當日範圍、ADR 比率
- 適用：指標圖表（建議日線）
- 參數：
  - `ADR_Period` (20): ADR 計算週期

### 3. UpsideDownside_Volatility.xs
**上漲/下跌波動率指標**
- 功能：計算分離的上漲和下跌波動率
- **這直接回答「平均上漲或下跌波動率是 +3% 或 -3%」的問題**
- 顯示：平均上漲 %、平均下跌 %、當日波動、淨波動
- 適用：指標圖表（建議日線）
- 參數：
  - `Period` (20): 計算週期

## 使用方式

### 在圖表上使用

1. 開啟 XQ 圖表
2. 選擇日線週期（建議）
3. 載入指標腳本
4. 查看繪製的指標線

### 參數調整

在腳本中修改 `input` 參數：
```xscript
input: ADR_Period(20, "ADR 計算週期");
```

或在 XQ 介面中調整參數設定。

## 指標說明

### ADR (Average Daily Range)
- **公式**：過去 N 日的平均 (High - Low)
- **用途**：判斷當日波動是否高於或低於平均
- **解讀**：
  - ADR 比率 > 1.2：高於平均波動
  - ADR 比率 0.8-1.2：正常波動
  - ADR 比率 < 0.8：低於平均波動

### ATR (Average True Range)
- **公式**：考慮跳空的真實波動範圍
- **用途**：更準確的波動率指標（特別是經常跳空的股票）
- **解讀**：類似 ADR，但考慮隔夜跳空

### 上漲/下跌波動率
- **公式**：
  - 平均上漲 = Average((High - Open) / Open * 100)
  - 平均下跌 = Average((Open - Low) / Open * 100)
- **用途**：了解股票在上漲和下跌方向的平均波動
- **解讀**：
  - 上漲 > 下跌：上漲方向更波動
  - 下跌 > 上漲：下跌方向更波動

## 範例輸出

### ADR_Indicator.xs
- Plot1: ADR 絕對值（例如：$3.50）
- Plot2: ADR 百分比（例如：2.30%）
- Plot3: 當日範圍（例如：$4.20）
- Plot4: 當日範圍百分比（例如：2.80%）
- Plot5: ADR 比率（例如：1.22x）

### UpsideDownside_Volatility.xs
- Plot1: 平均上漲波動率 %（例如：+3.20%）
- Plot2: 平均下跌波動率 %（例如：-2.80%）
- Plot3: 當日上漲波動率 %（例如：+2.50%）
- Plot4: 當日下跌波動率 %（例如：-1.80%）

## 注意事項

1. **資料週期**：建議使用日線圖表，指標會自動處理跨頻率資料
2. **所需 K 棒數**：腳本已設定 `settotalbar(250)`，確保有足夠歷史資料
3. **參數調整**：根據需要調整計算週期（14、20、30 日等）
4. **非日線圖表**：腳本會自動使用 `GetField()` 取得日線資料
