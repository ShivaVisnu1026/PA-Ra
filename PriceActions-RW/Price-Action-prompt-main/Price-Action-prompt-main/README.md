# 价格行为交易术语翻译

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## 简介

欢迎使用价格行为交易术语翻译项目！本项目`融合了约 80 多个专业术语`旨在为交易者提供准确、符合中国人阅读习惯的价格行为交易术语翻译，帮助您更好地理解和应用交易策略，提升交易水平。

本项目受阿尔·布鲁克斯（Al Brooks）的价格行为交易理论启发，专注于对盘面和每根 K 线的深入解读，确保翻译内容既专业又易于理解。官方网站：[https://www.brookstradingcourse.com/price-action-trading-terms-glossary/](https://www.brookstradingcourse.com/price-action-trading-terms-glossary/)

## 仓库由来

原本已经向沉浸式翻译官方仓库提交了 PR ，但由于沉浸式翻译被收购后官方太过于佛系没有合并，特意做此仓库以便大家使用

## 项目目标

- **准确翻译**：提供精准的价格行为交易术语翻译，确保不偏离原文含义。
- **符合习惯**：调整语句结构，使翻译内容更符合中国人的阅读习惯。
- **辅助学习**：帮助用户理解复杂的交易概念，提升交易决策能力。
- **清晰易读**：以简洁明了的方式呈现翻译内容，方便快速查阅。

## 术语列表预览

以下是部分价格行为交易术语的翻译，完整列表请参见 ([Price Action Trading.md](https://github.com/Stanley-baby/Price-Action-prompt/blob/main/Price%20Action%20Trading.md?plain=1)：

### K 线相关
- **bar**: K 线
- **body**: 实体
- **doji**: 十字星
- **tails**: 影线
- **BO/bo**: 突破

### 走势相关
- **MM**: MM
- **measured move**: 测量移动
- **pullback/PB/pb**: 回调
- **trend**: 趋势
- **swing**: 波段

### 交易相关
- **scalp**: 剥头皮交易
- **swing trade**: 波段交易
- **trader's equation**: 交易者等式
- **edge**: 先机/优势
- **setup**: 建仓形态

## 使用方法

您可以复制[Price Action Trading.md](https://github.com/Stanley-baby/Price-Action-prompt/blob/main/Price%20Action%20Trading.md?plain=1) 文件中的 prompt，然后在沉浸式翻译插件中使用。具体步骤如下：
1. 打开 [Price Action Trading.md](https://github.com/Stanley-baby/Price-Action-prompt/blob/main/Price%20Action%20Trading.md?plain=1) 文件
2. 复制文件中的 prompt 内容
3. 打开沉浸式翻译插件设置 -- AI 专家-- 下滑到页面最下方添加自定义专家
4. 将复制的 prompt 粘贴到System Prompt窗口替换默认 prompt
5. 保存并在沉浸式翻译侧边栏设置的 AI 专家中选择刚刚保存的 AI 专家即可

｜注：如需要免费使用 AI 服务可在插件中使用智谱 AI
